## 好视通私有云服务器一键安装脚本     
#由于4.37之后版本有官方的部署工具,故4.37之后只提供安装包下载,不再提供安装脚本    
#好视通服务器安装脚本6月1日起停止更新, 只做维护不在更新服务器版本    

脚本目前功能：  
1、同步linux时间  
2、检测linux硬件配置  
3、自动获取linux内外网IP地址  
4、一键安装CES标准版/CES中性版/FSP服务器/H323网关/录制服务器       
5、[RTMP/WebRTC/HLS/HTTP-FLV/SRT实时视频服务器](https://github.com/ossrs/srs)   
6、iperf3网络性能测试工具   
7、HTML5网络速度测试工具   
8、修改CES/FSP配置文件     
9、修改CES节点配置文件    
10、修改录制服务器配置文件   
11、修改H323服务器配置信息    
12、重置后台admin密码    
13、[ddns-go动态域名解析服务](https://github.com/jeessy2/ddns-go)    
14、一键部署Prometheus+Grafana+node-exporter+consul+alertmanager      

推流服务器采用了ossrs开源代码 
  
后续计划添加zabbix服务器监控应用  

一键安装脚本：     

bash <(curl -Ls https://ghproxy.com/https://raw.githubusercontent.com/e5sub/hst/master/ces.sh)
